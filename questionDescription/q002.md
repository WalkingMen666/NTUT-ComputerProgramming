# Q2. 求兩個方程式的實根

**一元二次方程式，ax^2 + bx + c = 0，輸入a, b, c, 求 方程式的兩個實根。**

## Instruction

> **輸入說明：**  
  **第一個數(int) a**  
  **第二個數(int) b**  
  **第三個數(int) c**  

> **輸出說明：**  
  **第一個實根 x1 = ((-b)+math.sqrt(b*b-4*a*c))/(2*a)**  
  **第二個實根 x2 = ((-b)-math.sqrt(b*b-4*a*c))/(2*a)**<br/>  
  **x1, x2 輸出到小數點第一位**  
  **print("%.1f" %x1)**

## [My Code](../HomeWork/q002.py)
```python
import math

a = int(input())
b = int(input())
c = int(input())

print(round((((-b) + math.sqrt(b*b-4*a*c)) / (2*a)), 1))
print(round((((-b) - math.sqrt(b*b-4*a*c)) / (2*a)), 1))
```
